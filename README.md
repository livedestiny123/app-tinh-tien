# App tính tiền - Ốc Hẻm Huế

Phần mềm bán hàng (POS) cho quán ốc: 30 bàn, tính bill, in bill,
sửa menu, thống kê doanh thu theo ngày, lịch sử ~30 ngày,
kết nối máy in nhiệt Bluetooth. Chạy được cả offline (PWA).

## Cài đặt
Mở qua đường link https (GitHub Pages hoặc Netlify),
rồi "Thêm vào màn hình chính" trên điện thoại.
